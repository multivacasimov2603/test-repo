## Un chiste para empezar:

**¿Por qué los programadores confunden la Navidad con Halloween?**

*Porque OCT 31 == DEC 25!*

```python
# En Python:
print(oct(31))  # '0o37' 
print(dec(25))  # '25'

# Pero en matemáticas:
octal_31 = 3*8 + 1 = 25
decimal_25 = 25

¡Son lo mismo! 🎃🎄
```
